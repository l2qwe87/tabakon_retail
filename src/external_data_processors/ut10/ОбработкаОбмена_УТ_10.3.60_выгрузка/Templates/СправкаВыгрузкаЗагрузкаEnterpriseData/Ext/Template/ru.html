Выгрузка и загрузка данных EnterpriseData

Обработка предназначена для выгрузки и загрузки данных с использованием универсального формата EnterpriseData. Предназначена для использования в информационных базах, поддерживающих синхронизацию данных через универсальный формат.

Работа обработки

После запуска обработки необходимо выбрать вид операции Загрузка или Выгрузка, после настройки всех параметров нажать кнопку Выполнить операцию.

Выполненные настройки можно сохранить и использовать при следующих сеансах работы (кнопки Сохранить настройки и Восстановить настройки).

Загрузка

Для загрузки данных необходимо указать Путь к файлу загрузки, содержащему данные для загрузки. Файл должен быть с версией формата, которая поддерживается данной обработкой.

Выгрузка

Для выгрузки данных необходимо:

  • Указать Путь к файлу выгрузки, в который будут выгружены данные
  • Указать Версию формата, в которой необходимо выгрузить данные. Рекомендуется использовать максимальную версию, поддерживаемую обоими конфигурациями (исходной и конечной).
  • Указать Данные для выгрузки. Если выбран Произвольный набор, настроить состав выгружаемых данных:
    1. Выбрать объекты метаданных
    2. При необходимости указать отбор по периоду для документов
    3. При необходимости настроить произвольные отборы для справочников и документов
  • Выгрузка данных, зарегистрированных на узле плана обмена. Для этих целей следует указать Узел обмена при выгрузке. В этом случае для настройки состава выгружаемых данных необходимо зарегистрировать объекты к обмену штатным функционалом.
